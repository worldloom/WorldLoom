# Code of Conduct

## Our Pledge

WorldLoom is committed to providing a welcoming and inclusive environment for everyone.

## Standards

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers.

All complaints will be reviewed and investigated promptly and fairly.
